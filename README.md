# Installation

1. Clone this repo:

```
git clone https://github.com/Dagen-Life/SmartContract
cd SmartContract
```

2. Install dependencies

```sh
yarn
```

3. Run the test suite (which also has all the functionality)

```
npx hardhat test
```
